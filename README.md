#💻 Quick Start
    1. Clone the repo:
      git clone https://github.com/VidhiAgrawa/React-Projects.git
    2. Navigate to a project folder:
      cd React-Projects/ProjectName
    3. Install and run:
      npm install
      npm start
    4. Now Visit: http://localhost:3000
