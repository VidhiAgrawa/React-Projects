##💻 Quick Start<br>
    1. Clone the repo:<br>
      git clone https://github.com/VidhiAgrawa/React-Projects.git<br>
    2. Navigate to a project folder:<br>
      cd React-Projects/ProjectName<br>
    3. Install and run:<br>
      npm install<br>
      npm start<br>
    4. Now Visit: http://localhost:3000<br>
